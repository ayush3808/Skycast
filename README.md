SkyCast – Weather App

A responsive weather forecasting web app that provides real-time weather, hourly updates, 5-day forecast, and air quality index using OpenWeatherMap APIs.

🛠️ Tech Stack
HTML5 – Structure
CSS3 – Styling & Responsive UI
JavaScript (Vanilla) – Logic & API handling
OpenWeatherMap API – Weather, Forecast, AQI
LocalStorage – Theme & Favorites
⚙️ Working
User searches a city or uses current location
App fetches current weather data (temperature, humidity, wind)
Then fetches:
Hourly forecast (24 hrs)
5-day forecast
Air Quality Index (AQI)
Data is dynamically updated in UI
Background & tips change based on weather conditions
Favorites and theme are stored using LocalStorage
